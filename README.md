# About
This repo is my AI-Assisted C++ Learning Path. I have zero experience with this programming language so I asked AI to be my instructor. Since my main interest is in Cybersecurity, the goal is take me from beginner to intermediate level by creating a Cyber Security Toolkit so I will actually have something useful at the end of the course.

Here are the toolkit features:

### Password Analyzer

- strength scoring
    
- pattern detection
    

### File Integrity Monitor

- tamper detection
    
- hashing
    

### Port Scanner

- TCP scanning
    
- multithreaded scanning
    

### Log Analyzer

- suspicious activity detection
    
- failed login analysis
    

### Encryption Utility

- file encryption
    
- SHA256 hashing
    

### Reporting Engine

- exportable reports
    
- threat summaries
